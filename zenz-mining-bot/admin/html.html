Module Initialization Overview

Purpose of __init__.py

This __init__.py file initializes the app module and prepares the application for use. It sets up the Flask application and starts the Telegram bot.

Key Functions

  • create_app() - Initializes and configures the Flask application.
  • bot.start() - Starts the Telegram bot to handle incoming commands.

Usage Instructions

  1. Clone the repository.
  2. Set up the virtual environment.
  3. Install the required packages.
  4. Start the Flask app and the Telegram bot.

Configuration

API keys and other configuration details should be added to config.py:
