# Telegram Mining Mini App  

This project is a Telegram bot that provides mining data from the Ethereum network and a web dashboard for users to view the information.  

## Requirements  

- Python 3.x  
- `python-telegram-bot`  
- `Flask`  
- `web3`  
- `pytest`
